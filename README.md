# base64-control-demo

A monorepo combining [python-experiments](https://github.com/spookyuser/base64alarm) and [skills-md](https://github.com/Augusta-Lina/skillsmd) as git submodules.

## Setup

```bash
git clone --recurse-submodules https://github.com/spookyuser/base64-control-demo.git
```

If you've already cloned without `--recurse-submodules`:

```bash
git submodule update --init --recursive
```
